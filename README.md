# 📚 Library Management System

一个的图书馆管理系统（C++ + SQLite3），区分 **书目（Book）** 与 **实体书（BookCopy）**，为后续功能扩展（借阅记录、推荐系统、多数据库支持等）打下良好基础。

---

## 🏗️ 系统架构概览(大概这样吧嘻)

```text
┌──────────────┐
│   UI / CLI   │   （后续可扩展 GUI / Web）
└──────┬───────┘
       │
┌──────▼───────┐
│  Service层   │   （业务逻辑，如借书、还书、推荐）
└──────┬───────┘
       │
┌──────▼───────────────┐
│        DAO 层         │
│  BookDAO / BookCopyDAO│
└──────┬───────────────┘
       │
┌──────▼──────────┐
│ DatabaseOperator │   （SQLite3 封装）
└──────┬──────────┘
       │
┌──────▼──────────┐
│     SQLite3     │
└─────────────────┘
```
